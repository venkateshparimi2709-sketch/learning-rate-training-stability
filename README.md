# Effect of Learning Rate on Neural Network Training Stability

Author: Venkatesh Parimi  

## Project Overview
This project presents a tutorial on how the learning rate influences the training stability of neural networks.  
The objective of this work is to explain both theoretically and experimentally how different learning rate values affect convergence behaviour, optimisation efficiency and model performance.

A simple neural network model is trained using different learning rate values on the Fashion-MNIST dataset.  
The results are analysed using visualisations such as training loss curves and validation accuracy plots.

## Contents of Repository
- Venkatesh_Parimi_Learning_Rate_Training_Stability.pdf  
  Tutorial report explaining theoretical concepts and experimental observations.

- Venkatesh_Parimi_learning_rate_training_stability.ipynb  
  Jupyter notebook containing the complete implementation and visualisation code.

## How to Run the Code
1. Open the notebook in Google Colab or Jupyter Notebook.
2. Run all cells in sequence.
3. The dataset will be automatically loaded.
4. Figures showing training stability behaviour will be generated.

## Learning Objectives
- Understand the role of learning rate in gradient-based optimisation  
- Analyse convergence behaviour of neural networks  
- Compare training performance under different hyperparameter settings  
- Gain practical experience in implementing neural network experiments
